HTML Lesson 3: Text Formatting, Ordered Lists, and Unordered Lists

Basic Text Formatting

Use the strong tag or the b tag to make something bold.

The Difference Between Strong and B

The Strong tag is an HTML Semantic tag. There is a meaning behind using the Strong tag. What that meaning is, it tells the browser there is something important about the text inside that tag.

The B tag means nothing. It simply bolds the text inside it.

The em Tag and the i Tag

The em Tag and the i Tag make the text inside of them italic. Again, the difference is the em tag is a semantic tag. Like the Strong tag, the em Tag is saying there is a reason the text inside is italic. The i Tag only turns things italic.

The u Tag

The u Tag simply underlines your text.
